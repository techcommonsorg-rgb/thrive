# Thrive@Meriden

Static website for the Thrive@Meriden student wellbeing proposal.

## Files

- `thrive-meriden.html`: main page content
- `index.html`: entry page used by GitHub Pages (redirects to `thrive-meriden.html`)

## Deploy on GitHub Pages

1. Open your repo on GitHub.
2. Go to **Settings** -> **Pages**.
3. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Save and wait 1-2 minutes.
5. Open the site URL shown in the Pages settings.

## Local Preview

Open `index.html` in your browser, or run a local static server from this folder.
